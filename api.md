# API Documentation

The OmniGate Commerce API provides comprehensive access to all platform features.

## Overview

OmniGate Commerce is built using the Gemini API with server-side integration for AI-powered e-commerce management.

## Configuration

### Environment Variables

Create a `.env` file in the project root with the following configuration:

```bash
GEMINI_API_KEY=your_api_key_here
```

See `.env.example` for a template.

## Capabilities

### Server-Side Gemini API

The application leverages server-side Gemini API integration for:

- Natural language processing
- Product recommendation
- Customer service automation
- Sales analytics and insights

## Mobile Application

The Android application provides:

- Real-time dashboard
- Order management
- Inventory tracking
- Analytics visualization
